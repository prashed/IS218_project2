Azure and Docker Setup

Step 1: Create an Azure account.

Step 2: Once logged in, go to Azure Portal

Step 3: Go to Create a resource > Containers > Container Registry. This will register a container for Docker to run through

Step 4: And after installing Azure SDK, your Visual Studio Server Explorer will be visible.

Step 5: Fill in required fields and create

Step 6: Once you have created the registry, we need to push our current docker image to the registry. To do this run the following commands

1. Build your image: docker build -t containerRegistryName.azurecr.io/imageName:latest .

2. Check if image created: docker images

3. Run the image locally: docker run -dp 80:80 containerRegistryName.azurecr.io/imageName

4. Log into azure container registry: docker login containerRegistryName.azurecr.io

5. Push the newly made image to the container registry: docker push containerRegistryName.azurecr.io/imageName

Step 7: Now that our docker image is on azure, we can display it using an App Service, so let's make one of those. Search for and create a Web App for Container
