What Is Pylint And How To Implement It

Pylint is a tool that parses your code and help you apply a universal format to your code to make it more readable. This can be stuff like adding docstrings to classes and functions, and also using good variable names and spacing.

Step 1: Install pylint by running "pip install pylint" in the python terminal. "Pip" is python's package manager that can be used to install different programs from Python's index.

Step 2: An example of a file made for my calculator program. However, I want to make sure my code is aligned with PEP8 standards.

Step 3: Go back to the python terminal and run "pylint (folderName)/(fileName)" or whatever file path that is suitable for your program.

Step 4: Pylint will inspect the code and grade it according to PEP8 standards, then it will produce a score based on how well it met those standards.
